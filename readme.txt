Add more theme location for menus.



== Description ==


you can add number of menu location in your theme with it.



== Installation ==



1. Upload `Menu Location` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Add menu location through the 'Settings => Menu Location' menu in wordpress



You're done!
